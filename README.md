# Agent Tribe One

This repo defines the **first tribe** for the Agent Tribes registry.

## Identity
- Source of truth: `tribe.yaml`

## Shared assets
- `skills/` — prompts / skills the tribe shares
- `resources/` — links, docs, tooling notes
- `corpus/` — curated texts for members
- `memes/` — culture / lightweight artifacts

## Publishing
Once you create the GitHub repo, update `links.repo` in `tribe.yaml`.
